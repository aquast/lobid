h1. Lobid

Linking Open Bibliographic Data.

This software implements the API of "http://lobid.org":http://lobid.org, the LOD service offered by "hbz":http://www.hbz-nrw.de.

For details about the data processing in our backend see "https://github.com/lobid/lodmill/":https://github.com/lobid/lodmill/.
